## Prerequisites

- **Bun** (recommended), Node.js 18+, or compatible runtime
- A database (PostgreSQL, SQLite, or MySQL)
- An ORM or driver supported by adapters (Drizzle, Prisma)

## Create the Engine

Create your conversation engine instance:

```ts title="lib/conversation.ts"
import { betterConversation } from "@better-conversation/core";
import { drizzleAdapter } from "@better-conversation/adapter-drizzle";
import { db } from "@/lib/db";

export const conv = betterConversation({
  adapter: drizzleAdapter(db, { provider: "pg" }),
});
```

## Configure Database

Run migrations to create the required tables. Choose your adapter:


  </CodeBlockTab>
  <CodeBlockTab value="prisma">

```ts
// Copy or extend schema from @better-conversation/adapter-prisma
```

  </CodeBlockTab>

Tables created: `bc_chatters`, `bc_conversations`, `bc_participants`, `bc_blocks`, `bc_chatter_permissions`, `bc_block_registry`, `bc_role_registry`, `bc_policies`, and related.

See [Database schema](/docs/schema) for full details.

## CLI (Schema Generation)

Use the CLI to generate Drizzle or Prisma schemas from your config:

```bash
bun add -D @better-conversation/cli
bunx bc generate --adapter drizzle
```

See [CLI documentation](/docs/cli) for generate and migrate commands.

## Mount the Handler

<CodeBlockTabs defaultValue="next">
  <CodeBlockTabsList>
    <CodeBlockTabsTrigger value="next">Next.js</CodeBlockTabsTrigger>
    <CodeBlockTabsTrigger value="hono">Hono</CodeBlockTabsTrigger>
    <CodeBlockTabsTrigger value="express">Express</CodeBlockTabsTrigger>
  </CodeBlockTabsList>
  <CodeBlockTab value="next">

```ts title="app/api/conversation/[[...path]]/route.ts"
import { conv } from "@/lib/conversation";
import { createNextHandler } from "@better-conversation/handler-next";

const handler = createNextHandler(conv, { basePath: "/api/conversation" });

export const GET = handler.GET;
export const POST = handler.POST;
export const PATCH = handler.PATCH;
export const DELETE = handler.DELETE;
```

  </CodeBlockTab>
  <CodeBlockTab value="hono">

```ts title="app.ts"
import { conv } from "@/lib/conversation";
import { createHonoHandler } from "@better-conversation/handler-hono";

app.route("/conversation", createHonoHandler(conv));
```

  </CodeBlockTab>
  <CodeBlockTab value="express">

```ts title="app.ts"
import { conv } from "@/lib/conversation";
import { createExpressHandler } from "@better-conversation/handler-express";

app.use(express.json());
app.use("/conversation", createExpressHandler(conv));
```

  </CodeBlockTab>
</CodeBlockTabs>

## Initialize the Engine

Before handling requests, call `init()` to register default blocks and roles:

```ts title="init.ts"
await conv.init();
```
