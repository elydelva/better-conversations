title: SSE

---

## Installation

```bash
bun add @better-conversation/plugin-sse
```

## Setup

```ts title="lib/conversation.ts"
import { betterConversation } from "@better-conversation/core";
import { createSsePlugin } from "@better-conversation/plugin-sse";

const conv = betterConversation({
  adapter,
  plugins: [createSsePlugin()],
});
```

## Endpoint

The plugin registers:

```
GET /conversations/:id/stream
```

This streams `block:created` events (and heartbeats) as new blocks are added to the conversation. The implementation is polling-based.

## Client Usage

```ts title="client.ts"
import { createConversationClient } from "@better-conversation/client";

const client = createConversationClient({ baseUrl: "/api/conversation" });

const es = client.stream(conversationId);

es.addEventListener("block:created", (e) => {
  const { block } = JSON.parse(e.data);
  console.log("New block:", block);
});

es.addEventListener("error", () => {
  es.close();
});
```

## Edge Compatibility

The SSE implementation is polling-based and Edge-compatible. No WebSocket server required.
