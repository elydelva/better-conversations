title: Audit

---

## Setup

Audit is built into the core. Enable it via `config.audit`:

```ts title="lib/conversation.ts"
import { betterConversation, createInMemoryAuditStore } from "@better-conversation/core";

const conv = betterConversation({
  adapter,
  audit: {
    store: createInMemoryAuditStore(),
  },
});
```

## Events

| Event | When |
|-------|------|
| `block:created` | After a block is sent |
| `conversation:created` | After a conversation is created |

## Custom Store

Provide a custom store for persistence:

```ts title="lib/audit.ts"
import type { AuditStore } from "@better-conversation/core";

const customStore: AuditStore = {
  async append(entry) {
    await db.auditLog.create({
      data: {
        event: entry.event,
        entityType: entry.entityType,
        entityId: entry.entityId,
        payload: entry.payload,
      },
    });
  },
  async query(filters) {
    return db.auditLog.findMany({
      where: {
        ...(filters.event && { event: filters.event }),
        ...(filters.entityType && { entityType: filters.entityType }),
        ...(filters.entityId && { entityId: filters.entityId }),
        ...(filters.since && { createdAt: { gte: filters.since } }),
      },
      take: filters.limit ?? 100,
    });
  },
};

betterConversation({
  adapter,
  audit: { store: customStore },
});
```

## Querying

`createInMemoryAuditStore()` returns a store with an `entries` array and optional `query` method:

```ts title="query.ts"
import { createInMemoryAuditStore } from "@better-conversation/core";

const store = createInMemoryAuditStore();
const conv = betterConversation({ adapter, audit: { store } });

// Access entries directly (in-memory store)
const entries = store.entries;

// Or use query when available
const recent = await store.query?.({
  event: "block:created",
  entityType: "block",
  since: new Date(Date.now() - 24 * 60 * 60 * 1000),
  limit: 100,
});
```
